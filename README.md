# Discord Role Setup Bot

A modal-based role setup bot for Discord.

## Setup
1. Add `DISCORD_BOT_TOKEN` to Railway environment variables
2. Deploy to Railway
3. Invite bot with permissions: Manage Roles, Send Messages, Embed Links

## Commands
- `!setup` - Send setup message (admin only)
- `!mydata` - Check your data
- `!cleardata @user` - Clear user data (admin)
